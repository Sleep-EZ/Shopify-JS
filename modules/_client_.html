ShopifyInstanceWrapper

ShopifyInstanceWrapper: object
  • Defined in client.ts:50

This type describes the response container that Shopify responds with in their JSON API. For example, requesting /products/some-product.json will return a request looking something like this:

    { "product": {"id": "1234567890", ...} }

This type allows us to describe the type (Generic T) contained inside each of the possible keys.
